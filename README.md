# my-dotfiles
I am using dwm in ArcoLinux.

- Lancher: [dmenu](https://github.com/AzmainBiswas/dmenu)

- Terminal: Alacritty, Kitty
- FetchTool: [fastfetch](https://github.com/fastfetch-cli/fastfetch)
- Filemannager: pcmanfm
- Image: sxiv
